General

VerdexSwap is a decentralized exchange (DEX) built on a custom EVM-compatible Proof-of-Authority blockchain (Chain ID 7201) with 5-second block times and near-zero gas fees. Users can swap tokens, provide liquidity, and earn yields with full self-custody of their assets. The ecosystem includes a DEX, block explorer, desktop wallet, and a testnet faucet.

The Verdex PRC20 Blockchain is a custom-built Layer 1 blockchain with Chain ID 7201. It uses a Proof-of-Authority (PoA) consensus mechanism to generate blocks every 5 seconds with minimal gas fees. It is fully EVM-compatible, meaning you can deploy Solidity smart contracts using Hardhat, Foundry, or Remix with zero configuration changes.

PRC20 is the Verdex token standard, functionally equivalent to ERC20 on Ethereum or BEP20 on BNB Smart Chain. PRC20 tokens follow the same interface (transfer, approve, balanceOf, etc.) but run natively on the Verdex blockchain, benefiting from faster confirmation times and lower transaction costs. Any ERC20-compatible wallet or tool works with PRC20 tokens.

Network & MetaMask

You can add the Verdex network to MetaMask in one click by visiting verdexswap.site/add-network.html. Alternatively, add it manually with these settings: Network Name: Verdex PRC20 Testnet, RPC URL: https://verdex-ecosystem-production.up.railway.app/rpc, Chain ID: 7201, Symbol: VDX, Explorer: https://verdexswap.site/explorer.

The Verdex PRC20 Testnet uses Chain ID 7201 (hex: 0x1C21). This is a unique, unregistered chain ID that does not conflict with any existing EVM network. The chain has been submitted to Chainlist.org for global listing (PR #8509).

The public RPC endpoint is https://verdex-ecosystem-production.up.railway.app/rpc. This endpoint supports standard Ethereum JSON-RPC methods including eth_chainId, eth_sendRawTransaction, eth_getBalance, eth_getBlockByNumber, and more. It runs 24/7 on cloud infrastructure.

Consensus & Security

Verdex uses a Proof-of-Authority (PoA) consensus mechanism where pre-approved validator nodes produce blocks every 5 seconds. Unlike Proof-of-Work (which requires mining) or Proof-of-Stake (which requires token deposits), PoA relies on the identity and reputation of validators. This allows for extremely fast block times, deterministic finality, and near-zero gas costs — ideal for a testnet and development environment.

Token & Economics

VDX is the native gas and governance token of the Verdex ecosystem. The total supply is fixed and distributed across liquidity incentives, team allocation (with vesting), ecosystem development, and community rewards. Detailed tokenomics including emission schedules, vesting timelines, and allocation percentages are documented in the Verdex Whitepaper.
