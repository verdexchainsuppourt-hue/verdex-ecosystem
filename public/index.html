Verdex Ecosystem Directory

Quick answers about the network, miner, faucet, and how to connect your wallet.

What is Verdex Swap and Blockchain Network?

Verdex is a next-generation green EVM blockchain network and decentralized exchange (DEX) engineered for eco-friendly DeFi transactions. Operating on a low-energy Proof-of-Authority (PoA) consensus mechanism (PRC20 standard), Verdex provides fast transactions, minimal gas fees, and a full self-custody ecosystem including swap pools, yield farms, and block explorers.

What is the Verdex DePIN Desktop Miner?

The Verdex DePIN desktop miner is a lightweight local application (available for Windows and Linux) that enables users to connect their idle GPU/CPU hardware power to the Verdex DePIN mining pool. By running the desktop miner, users secure the network infrastructure and earn Verdex Points (VP) automatically, which prepare them for the native VDX token distribution.

Is there a public faucet?

No. The testnet faucet is permanently retired. Mainnet VDX is never distributed through a public faucet. Mining rewards and P2P settlement use real VDX only after chain and contract verification.

What is the mainnet chain ID?

Proposed Verdex Mainnet chain ID is 72010 (Besu QBFT). Legacy testnet 7201 is retired. Add network only after /api/network returns verified. Explorer: https://verdexswap.site/explorer.
